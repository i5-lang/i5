# Contributing

* Create a fork of the repository.
* Clone the repo and switch to `dev` branch

```sh
go get -u github.com/i5/i5
cd $GOPATH/src/github.com/i5/i5/
git checkout dev
```

* Make changes
* Run tests

```sh
go test ./...
```
* Commit
* Push your changes to your fork
* Submit a pull request
