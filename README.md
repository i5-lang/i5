# The i5 programming language

The **i5** is a script programming language with a interpreter.

---

## Install

### Build from source

```sh
$ go get github.com/i5-lang/i5
```

### Download

* [https://github.com/i5-lang/i5/releases](https://github.com/i5-lang/i5/releases)

## Usage

```sh
 $ i5 file.i5
```
